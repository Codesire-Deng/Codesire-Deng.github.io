换底公式

$\log_ab = \frac{\ln b}{\ln a}$

证明：
$e^{\ln a * \log_an} = b$ 两边取对数，移项。

$A^{\log_bC} = C^{\log_bA}$

证明：两边取对数，得$\log_bC \ln A = \frac{\ln C}{\ln b}*\ln A$

分治的时间复杂度

$T(N) = aT(N/b) + \Theta(N^k \log_p N)$

$$
T(N) = \begin{cases}
O(N^{\log_b a})& \text{if }a > b^k \
O(N^k \log_{p+1} N)& \text{if }a = b^k \
O(N^k \log_{p} N)& \text{if }a < b^k \
\end{cases}
$$
